# TravelLogue - 旅行游记小程序

TravelLogue 是一个基于微信小程序平台开发的旅行游记分享应用，让用户可以记录和分享他们的旅行经历。

## 功能特性

- 📝 游记发布：创建和编辑旅行游记
- 👥 游记浏览：浏览其他用户的游记

## 技术栈

- 微信小程序原生开发
- Vant Weapp UI 组件库
- 瀑布流布局展示

## 项目结构

```
├── pages/                # 页面文件
│   ├── travelogue/      # 游记列表页
│   ├── community/       # 社区页面
│   ├── message/         # 消息页面
│   ├── travel/          # 行程页面
│   ├── our/            # 个人中心
│   ├── detail/         # 详情页面
│   ├── login/          # 登录页面
│   ├── edit/           # 编辑页面
│   └── add/            # 添加页面
├── images/             # 图片资源
├── server/             # 服务器端代码
└── miniprogram_npm/    # 小程序 npm 包
```

## 安装和运行

1. 克隆项目到本地
2. 安装依赖：
   ```bash
   npm install
   ```
3. 在微信开发者工具中导入项目
4. 构建 npm 包：
   - 在微信开发者工具中，点击 "工具" -> "构建 npm"
5. 编译运行项目
